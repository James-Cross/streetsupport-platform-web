# Street Support Network – Platform Web

This is the public-facing website for **Street Support Network**, helping people find local support services and access homelessness resources.

## Purpose

The site connects people experiencing or at risk of homelessness with verified local services. It also provides information for frontline workers, local authorities, and the public.

## Key Features

- Service search and filtering with geolocation support
- Location-specific pages for partnered regions
- Integrated virtual assistant for conversational support
- Resource library with downloadable materials
- Accessible and responsive design
